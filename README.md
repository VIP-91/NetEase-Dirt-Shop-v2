## 安装方法

1. 下载本地图的 .zip 文件
2. 解压得到世界文件夹（通常名为 `Netease_Dirt_Shop_v2` 或类似）
3. 将整个文件夹复制到 Minecraft 的 saves 目录：
   - Windows: `%appdata%\.minecraft\saves`
   - Mac: `~/Library/Application Support/minecraft/saves`
   - Linux: `~/.minecraft/saves`
4. 启动 Minecraft → 单人游戏 → 选择该地图即可游玩
